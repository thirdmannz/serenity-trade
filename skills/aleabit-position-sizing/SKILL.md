# Aleabit on Position Sizing

## Core principles

- **Build conviction first, size accordingly — or you'll capitulate the bottom and chase the top.** This is the single most-emphasized sizing rule.
- **Cap individual stock exposure around 5–10%** for most thematic positions. Don't go all-in on a small-cap thesis name.
- **Starter sizes in micro/small cap: 0.5–1%.** The SIVE entry was explicitly this small — room to scale on validation, not capitulation.
- **Highest-conviction concentration: ~30% in one name + blue-chip / broad ETF ballast.** This is the "barbell" — torque + survival.
- **Pair thematic small caps with a defensive core.** Don't stack a portfolio entirely with $400M-cap chokepoint plays; pair them with TSEM/SOI/COHR-tier "safest compounds" or broad ETFs.
- **Scaling is earnings-gated, not chart-gated.** Add on thesis validation (earnings, customer disclosures, supply data) — not on a green candle.

## How to apply

When a user asks "how much should I buy of X?", anchor on these tiers:

| Conviction tier | Position type | Sizing |
|---|---|---|
| Starter / scouting | Micro-cap with thesis still maturing | 0.5–1% |
| Standard thematic | Confirmed supply chokepoint, hyperscaler tie | 5–10% |
| Highest conviction | Validated by multiple earnings cycles | up to ~30%, balanced by blue-chip ballast |
| "Safest compounds" | Defensible large caps (TSEM/SOI/COHR-tier) | Ballast — can be larger combined |

Refuse to recommend "go all-in" sizing on a single small-cap thesis name — that's the explicit failure mode the principles guard against. If the user is using leverage, also pull in [[aleabit-risk-management]] for the leverage rules.
