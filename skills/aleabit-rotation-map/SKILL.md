# Aleabit on Institutional Rotation

## Core principles

- **Frontrun institutions, don't chase them.** The edge is buying a chokepoint name while it's still on Reddit/X, and exiting (or trimming) as institutional coverage / hedge fund accumulation arrives.
- **Rotation is sequenced, not random.** Capital moves through the AI capex stack node by node. If you map the sequence, the exit cue for one bucket is the entry cue for the next.
- **The 2nd-derivative signal is bank analyst behavior.** When sell-side gets passive-aggressive about a call he made, and then institutions start buying the *adjacent* name (IQE after AXTI), that's the rotation confirmation.

## How to apply

When the user asks "what's next?" or "where should I be looking?":

1. **Locate the user's current portfolio in the rotation map.** Are they in Phase 1/2 (mature) names? Then trimming + rotating into Phase 3 is the move. Already in Phase 3? Look at Phase 4 (power infra) for the next entry.
2. **Use the 2nd-derivative signal.** When sell-side analysts start covering a name he called early, that's the cue institutions are accumulating — trim, don't add.
3. **Match horizon to phase.** Phase 4 (just opening) is multi-year. Phase 3 (active) is 6–18 months until consensus. Phase 2 tail names are weeks-to-quarters of upside left.
4. **Don't fight the sequence.** Trying to bottom-fish a fully institutionalized Phase 1 name (memory) is not the same trade as the frontrunning trade — the asymmetry has already been priced.
